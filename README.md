# Simple Local File Upload & Download Server

A minimal Python server to upload and download files via your browser, running locally on your network.

## Features

- Upload files from your browser.
- Download files from a simple file list.
- Easy to run, no extra dependencies.
- Works from any device on your Wi-Fi network.

## Usage

1. Run `python 3d.py` 
2. Access the server via the printed URL 
3. Upload or download files through the web interface.

## Notes

- All uploaded files are saved in the `uploads` folder.
- Tested on Python 3.7+.
